<h1> Table Attributes </h1>
<font size=5>
You can use the following attributes in table
&#60table&#62 tags:
<p>
<ol>
<li> align= </li>
<li> bgcolor= </li>
<li> border= </li>
<li> bordercolor= </li>
<li> cellpadding= (padding around content) </li>
<li> cellspacing= (space between cells) </li>
<li> width </li>
</ol>
<table align=center bgcolor=purple border=5 bordercolor=blue cellpadding=20 cellspacing=10 width=50%>
<tbody align=center>
<tr>
<td bgcolor=white><font size=6> hello </td>
<td bgcolor=red><font size=6> hello </td>
<td bgcolor=purple><font size=6> hello </td>
</tr>
<table>
<hr>
You can also add these attributes to cells
&#60td&#62
<ol start=8>
<li> colspan= (merges a cell across columns) </li>
<li> rowspan= (merges a cell down columns)</li>
</ol>
<table align=center bgcolor=purple border=5 bordercolor=blue cellpadding=20 cellspacing=10 width=50%>
<tbody align=center>
<tr>
<td colspan=4> Row 1 </td>
</tr>
<tr bgcolor=teal>
<td> row 2 col 1 </td> 
<td> row 2 col 2 </td> 
<td> row 2 col 3 </td> 
<td> row 2 col 4 </td> 
</tr>
<tr bgcolor=grey>
<td rowspan=2> row 3 col 1 </td>
<td> row 3 col 2 </td>
<td> row 3 col 3 </td>
<td> row 3 col 4 </td>
</tr>
<tr bgcolor=blood>
<td> row 4 col 1 </td>
<td> row 4 col 2 </td>
<td> row 4 col 3 </td>
</table>
